## 설치

1. `chrome://extensions` 접속
2. "개발자 모드" 활성화
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. 이 폴더 선택

## 사용법

1. YouTube 영상 페이지에서 사이드바의 "스크립트 보기" 버튼 클릭
2. 원본/번역 버튼으로 전환
3. 타임스탬프 체크박스로 시간 표시 여부 선택
4. 복사 버튼으로 클립보드에 복사
5. 긴 영상은 구간별 복사 버튼으로 원하는 구간만 복사

## 설정

확장 프로그램 아이콘 우클릭 → 옵션

- **Gemini API Key**: 번역 기능 사용을 위한 API 키
- **번역 프롬프트**: `{text}` 자리에 원본 스크립트가 들어감
- **구간 복사 단위**: 스크립트를 몇 분 단위로 나눌지 설정 (기본 30분)
